# 🐤 Flappy Bird Game (Java)

A simple Flappy Bird clone built using Java and Swing.  
This project replicates the classic Flappy Bird gameplay where the player controls a bird, avoiding obstacles (pipes) to score points.

---

## 🎮 Features

- Smooth bird movement and gravity mechanics
- Randomly generated pipes
- Collision detection
- Score tracking system
- Game over and restart functionality
- Simple and clean UI using Java Swing

---

## 🛠️ Technologies Used

- Java
- Swing (for GUI)
- AWT (for graphics and event handling)
